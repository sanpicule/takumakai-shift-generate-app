バージョン 1.0.0 ／ 所要時間 約 5 分

# 1. ご利用前のご確認

インストールを始める前に以下をご確認ください。

- 対象 OS：**Windows 10 / Windows 11（64bit）**
- インターネットに接続されていること
- パソコンの**管理者権限**でログインしていること
- 空き容量が **500 MB 以上** あること

<div class="note">

過去のバージョンをお使いの場合は、念のため事前にデータのバックアップを取っておくことを推奨します（保存場所は後ほどご案内します）。

</div>

---

# 2. インストーラーのダウンロード

1. ブラウザ（Edge / Chrome など）を起動します
2. 配布担当者よりお伝えしている **ダウンロードページの URL** を入力します
3. ページ内の「**Assets（アセット）**」を開きます
4. 以下のファイル名のものをクリックしてダウンロードします

```
shift-scheduler-1.0.0-setup.exe
```

<div class="note">

「このファイルは一般的にダウンロードされていません」と表示された場合は、「**保持する**」を選択してください。

</div>

---

# 3. インストールの実行 ①  ― 警告画面の対応

ダウンロードしたファイルをダブルクリックすると、青い画面が表示される場合があります。

**「Windows によって PC が保護されました」** が出たら：

1. 画面中央の「**詳細情報**」をクリック
2. 下に現れる「**実行**」をクリック
3. 「ユーザーアカウント制御」が出たら「**はい**」をクリック

<div class="warn">

この警告は、**署名のついていないアプリで必ず表示される一般的なもの**です。本アプリは院内利用専用に開発されており、安全にご利用いただけます。

</div>

---

# 4. インストールの実行 ②  ― ウィザードの進行

セットアップウィザードが起動します。

1. 「**次へ**」をクリックして画面を進めます
2. インストール先フォルダーは **原則そのまま** で OK
3. 「**インストール**」をクリック
4. 完了画面が表示されたら「**完了**」をクリック

<div class="note">

インストールが終わると、自動的に **デスクトップにショートカット** が作成されます。

</div>

---

# 5. アプリの起動

以下のいずれかの方法で起動できます。

- デスクトップの「**看護シフト生成**」アイコンをダブルクリック
- スタートメニューから「**看護シフト生成**」を検索してクリック

<div class="note">

初回起動時は画面が表示されるまで **数秒〜十数秒** かかる場合があります。少々お待ちください。

</div>

---

# 6. アップデート（バージョンアップ）

本アプリは**自動更新には対応していません**。
新バージョンのご案内が届いたら、以下の手順で更新してください。

1. アプリを終了する
2. 同じ手順で**最新版インストーラーをダウンロード**
3. そのままインストールを実行（旧版は自動で置き換わります）

<div class="note">

入力済みの **スタッフ情報・シフトデータは更新後もそのまま引き継がれます**。再入力は不要です。

</div>

---

# 7. アンインストールとデータの保存場所

## アンインストール手順
スタートボタン右クリック → 「インストールされているアプリ」 → 「**看護シフト生成**」 → 「**アンインストール**」

## データの保存場所

```
C:\Users\<ユーザー名>\AppData\Roaming\shift-scheduler\
```

エクスプローラーのアドレスバーに `%APPDATA%\shift-scheduler` と入力すると直接開けます。

<div class="warn">

このフォルダーには **スタッフ氏名等の個人情報が含まれます**。院外パソコンへのコピー・持ち出しは行わないでください。

</div>

---

# 8. よくあるご質問・お問い合わせ

| 症状 | 対応方法 |
|------|---------|
| ダブルクリックしても起動しない | ダウンロードが未完了の可能性あり。再ダウンロード |
| 警告画面で進めない | スライド「3. インストールの実行 ①」を再確認 |
| インストール後に起動しない | パソコン再起動後に再試行 |
| データが消えた | 「7. データの保存場所」のファイルを確認 |
