# 看護シフト生成アプリ

卓麻会 病棟看護師の月間シフトを自動生成するデスクトップアプリ（macOS / Windows 対応）

---

## セットアップ・起動手順

### 1. 初回セットアップ（1回だけ）

ターミナル（macOS）またはコマンドプロンプト（Windows）を開き、このフォルダに移動して実行：

```bash
npm install
```

### 2. 開発版で起動（動作確認・日常利用）

```bash
npm run dev
```

### 3. 配布用ビルド（インストーラー作成）

```bash
# macOS用（.dmg）
npm run build:mac

# Windows用（.exe インストーラー）
npm run build:win

# 両方同時
npm run build:all
```

ビルド完了後、`dist/` フォルダにインストーラーが生成されます。

---

## アプリの使い方

### 手順

1. **シフト生成** タブで対象月・前月情報を入力
2. **スタッフ管理** タブでスタッフの勤務種別を設定（初回のみ）
3. **希望休入力** タブでカレンダーから希望休をタップ選択
4. **シフト生成** タブに戻り「生成」ボタンを押す
5. **シフト結果** タブで結果を確認・出力

### 出力方法

- **「スプレッドシートにコピー」** → クリップボードにコピー → Googleスプレッドシートに貼り付けるとそのままシフト表になる
- **「CSV保存」** → CSVファイルを保存 → ExcelやGoogleスプレッドシートで開く
- **「MD形式コピー」** → マークダウンテーブル形式でコピー

## シフト記号

| 記号 | 意味 | 時間 |
|------|------|------|
| 日 | 日勤 | 8:30〜17:00 |
| 当 | 当直 | 16:30〜翌9:00 |
| 明 | 明け（当直翌日） | 勤務免除 |
| 休 | 公休 | — |
| 代 | 代休 | 土日当直・日曜日勤の補償 |
| 希 | 希望休 | — |
| （空欄） | 人員不足（手入力） | — |

---

## 適用ルール（R1〜R16）

- **R1〜R4**（必須）: 毎日当直1名・平日/土曜日勤2名・日曜日勤1名
- **R5**（必須）: 当直翌日は必ず明け
- **R6**: 5連勤回避（当直専従で6連勤許可設定ありの場合のみ6連勤可）
- **R8**: 当直→明け→休みのサイクル
- **R11**: 土曜当直・日曜当直・日曜日勤に代休付与
- **R13**: 月間公休 = 土曜日数 + 日曜日数
- **R14**: 希望休を最優先反映

制約を全て満たせない場合は優先度の低いものから緩和し、レポートに記録されます。
